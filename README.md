Une description concise du projet et ses objectifs sans jargon excessif :

Second year enginering student project in python aiming to learning how to manipulate data, cleaning them and using them to create dashboards.

Les prérequis avant utilisation (versions, dépendances) :

This project need internet to works in order to download data and modules
I will recommend to download minimum Python 3.10 or plus to be compatible with all modules.
You will find all modules required for the programm in the file "requirement.txt". To download all the modules in one command in the shell with this command : "python -m pip install -r requirements.txt".

Un guide d’installation détaillé :

If you satisfied all the requirements specified upper, you will be able to launch main.py
When main.py will be launch a pop up will suggest u to launch the app on the http://127.0.0.1:8051/. 
If the pop up doesn't appear I suggest u to open a navigator and entering the link upper.
